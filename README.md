# in-memory-database-system
## Fase 1: Representacion de datos
    1.1 Sistema de tipos de datos:
    Desarrollar un sistema que pueda representar diferentes tipos de valores que una base de datos necesita almacenar.
### Que debe de hacer:
    - Soportar al menos 5 tipos básicos: números enteros, números decimales, texto, booleanos, y valores nulos
    - Permitir almacenar cualquiera de estos tipos en la misma estructura
    - Permitir comparar valores del mismo tipo entre sí
    - Permitir convertir valores a formato de bytes para almacenamiento
    - Permitir reconstruir valores desde bytes
